# Palambur

Outils et méthodes d'IA appliquée pour les **PME de Suisse romande** — utiles en atelier comme au bureau, avec les garde-fous du droit suisse.

## Projet en avant

**[palambur-method](https://github.com/PALAMBUR/palambur-method)** — un skill [Claude Code](https://docs.claude.com/en/docs/claude-code) qui **pose les bonnes questions avant de produire** un livrable métier (offre d'emploi, relance client, devis, note interne…). Il rend deux choses : le prompt réutilisable rempli, puis le livrable final rédigé. Protection des données par défaut (anonymisation, mention nLPD), aucune donnée inventée.

## Principes

- **Conformité** — nLPD, LEg (rédaction inclusive h/f/d), CO/LTr intégrés aux livrables.
- **Sobriété** — pas de promesse magique : toute donnée manquante reste un marqueur visible, et la validation humaine est rappelée sur les livrables sensibles.
